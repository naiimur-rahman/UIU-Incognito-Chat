## üìñ Overview

**Incognito Chat** is a modern, responsive web application designed for the UIU community. It bridges the gap between anonymous interaction and secure member communication. 

Built with a glassmorphism UI, it features a custom **AI Personality ("BhodAi")** powered by Google Gemini, robust image moderation via Cloudinary/Amazon Rekognition, and real-time syncing via Firebase.

## ‚ú® Features

### ü§ñ AI & Automation
* **"BhodAi" Integration:** A sarcastic, witty AI chatbot (powered by Google Gemini 2.5) that understands Bangla/English and UIU context.
* **Smart Moderation:** Automatic detection of inappropriate images (Nudity, Violence, etc.) using Amazon Rekognition via Cloudinary.

### üí¨ Chat Experience
* **Dual Login Modes:** * **Guest:** Quick, anonymous access with a secret name.
    * **Member:** Secure login with password, avatar storage, and history.
* **Rich Media:** Image uploads (supports HEIC conversion), Emojis, and Message Reactions.
* **Swipe to Reply:** Mobile-friendly gesture support for replying to specific messages.
* **Dark/Light Mode:** Fully responsive theme toggling with local storage persistence.

### üõ°Ô∏è Administration
* **Message Pinning:** Admins can pin important announcements to the top.
* **Moderation Tools:** Admins can delete messages instantly.
* **Status Indicators:** Real-time online user count.
